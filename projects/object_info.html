사물 형상 정보 추정 개선

# Object Detection # Top 1 Accuracy # Depth map

Background

사용자가 원하지 않는 사물에 대한 콘텐츠가 노출되거나 콘텐츠 배치가 어색한 문제를 해결 합니다.

Goals

  • ● 사물 검출 TOP 1 선택 로직 개선
  • ● 깊이맵, 딥러닝 기반 사물 영역 종합 로직 개발

Contents

사물 검출 TOP 1 선택 로직 개선

  ● 스캔 가이드 크기 변화에 따른 IoU, IoP 측정 → 최적의 스캔 가이드 크기 도출

  ● 스캔 가이드 내 사물 크기에 따른 최종 1개 사물 선택 로직 개발

깊이맵, 딥러닝 기반 사물 영역 종합 로직 개발

  ● 주변 환경에 따라 깊이맵 기반이 유리한 상황 존재

Result

서비스에 가장 적합한 스캔 가이드 영역을 찾고 Top 1 정확도가 향상됨에 따라 유저가 다른 사물이 선택되어 다시 스캔하는 케이스를 10% 줄일 수 있었습니다. 또한 사물에 해당하는 영역이 이전보다 정교하여 사물에 더 핏하게 콘텐츠를 증강시킬 수 있습니다.
